# STM32 Hệ thống khí nén HCMUTE

## Description

STM32F103VCT6 Hệ thống khí nén HCMUTE sử dụng thư viện HAL

## Features

- UART Communication
- GPIO Control
- Timer Interrupt
- PWM Output
- Embedded Programming in C

## Development Environment

- KeilC
- STM32 CubeMX
- STM32 HAL Library
- C Programming Language
